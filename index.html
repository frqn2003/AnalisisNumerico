Características principales
  • Interfaz moderna en modo oscuro, amigable y fácil de usar.
  • Resolución de ecuaciones no lineales y sistemas lineales.
  • Visualización gráfica de funciones y soluciones.
  • Paso a paso visual de los métodos.
  • Exportación de gráficos como imagen o GIF animado.
  • Teoría y explicación de cada método incluida.
  • Historial de cálculos recientes.
Métodos incluidos
  • Bisección: Encuentra raíces dividiendo el intervalo.
  • Newton-Raphson: Aproximación rápida usando derivadas.
  • Secante: Método sin derivadas, usando dos puntos.
  • Regula Falsi: Variante de bisección con mejor convergencia.
  • Gauss-Jordan: Resolución directa de sistemas lineales.
  • Jacobi: Método iterativo para sistemas.
  • Gauss-Seidel: Iterativo, mejora Jacobi usando valores recientes.
¿Cómo instalar?
  1. Descarga el archivo Métodos Numéricos APK usando el botón de arriba.
  2. Copia el archivo a tu dispositivo Android si lo descargaste en PC.
  3. En tu Android, abre el archivo descargado.
    Si es la primera vez, permite la instalación de apps desde fuentes desconocidas.
  4. Sigue las instrucciones en pantalla para instalar la app.
Requisitos: Android 5.0 o superior.
Nota: Si tienes problemas con la instalación, revisa los permisos de tu dispositivo.
